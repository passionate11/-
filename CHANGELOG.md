# Changelog

## 0.1.12

- 修复非强提醒休息页仍像“固定在最前面”的问题：默认休息页保持普通层级，应用失焦时会主动让开。
- 非强提醒休息页点击卡片空白或背景都会隐藏，只保留完成、稍后、跳过和延长按钮响应点击。
- 休息页弹出时不再关闭设置窗口，窗口诊断新增 key/main 状态，便于排查层级问题。
- 菜单栏新增 `复制自动化诊断`，聚合 URL Scheme、常用联动链接、轻打扰状态、前台应用和策略数量。
- 外部自动化新增 `songyixia://automation/diagnostic`，可脚本化复制自动化排查信息。
- 冒烟测试新增自动化诊断剪贴板校验，方便排查快捷指令、Raycast 或 Hammerspoon 联动问题。

## 0.1.11

- `复制专注联动脚本` 新增 Raycast Script Command 示例，覆盖切换轻打扰和暂停 30 分钟。
- 冒烟测试新增 Raycast 模板校验，确认复制出的专注联动模板包含 Raycast 配置片段。

## 0.1.10

- 菜单栏新增 `复制专注联动脚本`，一键复制 macOS 快捷指令和 Hammerspoon 可用的轻打扰联动模板。
- 外部自动化新增 `songyixia://automation/focus-template`，可直接把专注联动模板复制到剪贴板。
- 冒烟测试新增剪贴板校验，确认模板包含 `songyixia://focus/on`、`songyixia://focus/off` 和 Hammerspoon 示例。

## 0.1.9

- 菜单栏新增 `运行显示恢复压测`，用于模拟休息页跑到屏幕外的显示器切换场景。
- 外部自动化新增 `songyixia://diagnostics/display-recovery`，可脚本化验证外接屏断开或多显示器切换后的窗口恢复。
- 冒烟测试新增显示恢复校验，确认屏幕外休息页会回到当前屏幕内并清理测试状态。

## 0.1.8

- 菜单栏新增 `运行午休恢复压测`，专门模拟长时间离开后站立休息已经过期的场景。
- 外部自动化新增 `songyixia://diagnostics/lunch-recovery`，可由脚本、快捷指令或测试流程触发午休恢复压测。
- 冒烟测试新增午休恢复校验，确认过期站立休息会自动结算、关闭休息页并重置下一次站立计时。

## 0.1.7

- 修复默认休息页像“固定在最前面”的问题：非置顶模式不再主动抢焦点，也不再跨所有桌面显示。
- 非置顶休息页点击卡片外背景或切到其他 App 时会自动让开，本轮休息计时继续，不会下一秒又弹回前台。
- 应用诊断新增休息页层级、窗口行为和“已让开”状态，方便排查窗口挡住主屏幕的问题。

## 0.1.6

- 菜单栏新增 `快速节奏`，可一键切换 `20-20-20`、`番茄 25/5` 和 `调试 10 秒`。
- 当前节奏会在快速节奏菜单里打勾，切换后会自动保存并重新开始眼睛休息计时。
- 外部自动化新增 `songyixia://rhythm/debug`、`songyixia://rhythm/pomodoro` 和 `songyixia://rhythm/202020`。

## 0.1.5

- 菜单栏新增 `反馈问题...`，可直接打开 GitHub Issue，并预填版本、系统和安装路径。
- Issue 模板会提示先复制应用诊断，方便反馈窗口置顶、卡住、计时或自动化问题。

## 0.1.4

- 休息窗口默认不再强制置顶，切到其他 App 后不会一直压在最前面。
- 设置页和菜单栏新增 `置顶强提醒`，需要严格打断时可手动开启。
- 应用诊断新增置顶强提醒状态，方便排查窗口层级问题。

## 0.1.3

- `检查更新...` 改为真实检查 GitHub 最新 Release，并提示是否已经是最新版本。
- 更新检查失败时提供打开下载页兜底。

## 0.1.2

- 菜单栏新增 `复制应用诊断`，可一键复制版本、计时、设置、自动化、窗口和恢复事件摘要。
- 方便反馈卡住、置顶、多实例、提醒异常等问题时直接附带状态信息。

## 0.1.1

- 菜单栏新增 `检查更新...`，可直接打开 GitHub 最新 Release 页面。
- 关于/更新入口更完整，方便已安装用户找到源码和新版下载包。

## 0.1.0

首个可运行的 macOS 菜单栏版本。

- 眼睛休息提醒：支持 20-20-20、番茄钟和自定义分钟/秒。
- 站立提醒：眼睛和站立分别独立计时，默认每 2 小时提醒一次，站立 20 分钟。
- 全屏休息页：支持完成、稍后、跳过、延长和 Esc 应急跳过。
- 多风格 UI：极简呼吸、松弛森林、像素窗边、软糖玩具、夜间护眼，风格同时影响设置页和休息页。
- 设置窗口：macOS 风格左侧导航，覆盖眼睛、站立、显示、自动化和统计。
- 自动化：支持轻打扰、会议/演示/视频/游戏识别、日历会议、安静时段和 `songyixia://` 外部链接。
- 统计与备份：本机 7 天趋势、30 天摘要、热力图、CSV 导出和 JSON 备份/恢复。
- 恢复诊断：支持睡眠/锁屏/唤醒后状态校准、恢复压测和本机诊断脚本。
- 分发：生成 AppIcon、本地 ad-hoc 签名、zip 打包、安装到 `/Applications/松一下.app`，并通过 GitHub Actions 构建校验。
