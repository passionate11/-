# 松一下

松一下是一个轻量 macOS 菜单栏休息提醒工具，覆盖眼睛远眺、站立活动和番茄休息。

## 功能

- 眼睛休息提醒：默认 20-20-20，使用电脑 20 分钟后，提醒眺望 6 米外至少 20 秒。
- 站立提醒：默认每 2 小时提醒一次，站立 20 分钟，并支持均衡活动、肩颈舒展、走动循环、恢复放松四套动作组合和轻柔/标准/活动一点三档强度。
- 眼睛提醒和站立提醒分别独立计时。
- 支持番茄钟预设：25 分钟专注，5 分钟休息。
- 设置窗口支持直接输入分钟和秒。
- 菜单栏支持快速节奏，可一键切换 20-20-20、番茄 25/5 和调试 10 秒。
- 菜单栏可实时显示眼睛提醒和站立提醒，也支持智能轮换的短文案模式。
- 站立完成后会在菜单栏短暂显示完成反馈，方便确认本次流程已经结束。
- 菜单栏会显示最近一次睡眠、锁屏、解锁或屏幕变化后的恢复诊断，并跨应用重启保留最近事件链。
- 菜单栏支持午休恢复压测，可模拟长时间离开后站立休息过期并验证自动结算。
- 菜单栏支持显示恢复压测，可模拟外接屏断开或多显示器切换后休息页跑到屏幕外的场景。
- 支持暂停、立即眼睛休息、立即站立、重新开始全部计时。
- 支持独立设置窗口，可详细配置两个提醒、通知、提醒窗口和置顶强提醒。
- 设置窗口采用更接近 macOS System Settings 的左侧导航布局。
- 到点可弹出休息页，并发送系统通知；默认不强制置顶，点击卡片外背景或切到其他 App 时会让开，需要严格打断时可开启置顶强提醒。
- 支持多种整体风格：极简呼吸、松弛森林、像素窗边、软糖玩具、夜间护眼。
- 风格会同时影响设置窗口和全屏休息页。
- 显示设置页支持风格预览，切换风格时可以先看到休息页的大致气质。
- 支持自动轻打扰：会议、演示、视频或游戏类应用在前台时，不弹全屏休息页，只保留系统通知和计时。
- 支持日历会议降打扰：授权后，当前日历事件进行中时自动只发通知，不弹全屏休息页。
- 支持日程关键词策略：录制、直播、面试等日程可自动暂停计时，普通会议继续只发通知。
- 菜单栏可复制真实日历诊断，查看当前授权、进行中的日程和实际命中的日历策略。
- 菜单栏可运行真实日历联动自检，在当前真实会议或日程中验证到点提醒是否按策略不弹休息页。
- 支持全屏/演示降打扰：进入全屏、隐藏 Dock 或隐藏菜单栏等演示状态时自动只发通知。
- 支持安静时段：用户可设置固定时间段自动只发通知、不弹全屏休息页，用公开能力覆盖固定勿扰/专注场景。
- 支持外部自动化链接：快捷指令、Raycast、Hammerspoon 等工具可打开 `songyixia://focus/on`、`songyixia://rhythm/debug` 等链接来联动专注/勿扰和快速节奏。
- 菜单栏可复制专注联动脚本，直接生成快捷指令、Hammerspoon 和 Raycast 可用的轻打扰模板。
- 菜单栏可复制自动化诊断，方便排查快捷指令、Raycast 或 Hammerspoon 联动问题。
- 支持自定义轻打扰应用白名单。
- 支持按应用和日程设置自动化策略：只发通知、自动暂停、或不处理。
- 支持菜单栏显示模式：眼睛 + 站立、只显示眼睛、只显示站立、极简图标、智能轮换。
- 支持登录时自动启动。
- 设置窗口包含本机 7 天休息趋势、近 30 天摘要、30 天热力图、跳过率、站立总时长、连续天数、自动化策略占比、CSV 导出、JSON 备份和恢复。
- 菜单栏支持查看关于信息，显示版本、构建号、安装位置和 GitHub 源码入口。
- 菜单栏支持检查更新，可检查 GitHub 最新 Release 并打开下载页。
- 菜单栏支持复制应用诊断，方便反馈计时、窗口、自动化和恢复状态。
- 菜单栏支持反馈问题，可打开 GitHub Issue 并预填版本、系统和安装路径。
- 设置会自动保存到 `UserDefaults`。

## 构建

```bash
scripts/build_app.sh
```

构建产物会生成到：

```text
outputs/EyeRest.app
```

GitHub Actions 会在 `main` 和 Pull Request 上运行 macOS 构建检查，覆盖 shell 语法、App bundle 生成、URL Scheme 和关键入口 selector。

构建时会自动生成 `AppIcon.icns` 并写入 App bundle。

本地构建会默认使用 macOS ad-hoc 签名，方便安装和打包后做完整性校验，不需要 Apple Developer 账号。需要跳过签名时可以运行 `SKIP_CODESIGN=1 scripts/build_app.sh`；需要使用正式证书时可以运行 `CODESIGN_IDENTITY="Developer ID Application: ..." scripts/build_app.sh`。

## 打包

```bash
scripts/package_app.sh
```

打包产物会生成到：

```text
dist/songyixia-<version>-<build>.zip
```

解压后得到 `松一下.app`。

GitHub Actions 也会上传同名构建 artifact，方便直接下载测试。

应用版本号来自仓库根目录的 `VERSION` 文件，构建号可通过 `BUILD_NUMBER` 环境变量覆盖。

打包前会重新构建并签名，CI 会验证 zip 解压后的 `松一下.app` 仍然通过 `codesign --verify --deep --strict`。

版本更新内容见 [CHANGELOG.md](CHANGELOG.md)。

## 发布到 GitHub

确认 `VERSION` 和 [CHANGELOG.md](CHANGELOG.md) 已更新后，先跑：

```bash
scripts/preflight_release.sh
```

再推送版本标签：

```bash
git tag v$(cat VERSION)
git push origin v$(cat VERSION)
```

GitHub Actions 会在 `v*` 标签上再次运行发布前检查，并把 `dist/songyixia-<version>-<build>.zip` 上传到 GitHub Releases。标签版本必须和 `VERSION` 完全一致。

## 运行

```bash
open outputs/EyeRest.app
```

打开后点击菜单栏里的眼睛图标，选择 `打开设置...` 可以进入详细设置窗口。

菜单栏里的 `关于 松一下...` 会显示当前版本、构建号、安装位置和 GitHub 仓库入口。

菜单栏里的 `检查更新...` 会检查 GitHub 最新 Release，并在发现新版时提示打开下载页。

菜单栏里的 `复制应用诊断` 会把版本、计时、设置、自动化、窗口和恢复事件摘要复制到剪贴板。

菜单栏里的 `快速节奏` 可以直接切换 `20-20-20`、`番茄 25/5` 和 `调试 10 秒`。当前命中的节奏会有对勾，切换后会保存设置并重新开始眼睛休息计时。

菜单栏里的 `复制专注联动脚本` 会把常用 `songyixia://` 链接、快捷指令说明、Hammerspoon 示例和 Raycast Script Command 示例复制到剪贴板。

菜单栏里的 `复制自动化诊断` 会把 URL Scheme、常用联动链接、轻打扰状态、前台应用和策略数量复制到剪贴板。

菜单栏里的 `反馈问题...` 会打开 GitHub Issue，并预填当前版本、系统和安装路径。遇到卡住、置顶、计时或自动化异常时，先点 `复制应用诊断`，再把诊断粘贴到 Issue 里。

菜单栏里的 `置顶强提醒` 默认关闭。关闭时，休息窗口保持普通层级，不会长期压在其他 App 前面；点击休息页卡片空白、背景，或切到其他 App 时会让它隐藏。打开后，休息页会用更强的置顶层级提醒你离屏。

如果 macOS 提示来自未验证开发者，可以在 Finder 中右键打开，或到系统设置里允许打开。

```bash

```

## 本机诊断

```bash
scripts/diagnose_app.sh
```

脚本会读取 `/Applications/松一下.app` 的 bundle 信息、签名状态、运行进程、偏好设置摘要和可用的窗口列表，不会修改应用状态。遇到窗口置顶、休息页卡住、多个进程、安装或签名异常时，可以先运行它并把输出贴出来。

也可以检查未安装的构建产物：

```bash
APP_TARGET=outputs/EyeRest.app scripts/diagnose_app.sh
```

## 说明

当前可运行版本使用 Objective-C + AppKit 实现，以便在只有 Command Line Tools、没有完整 Xcode 的环境中构建。`Sources/EyeRest/main.swift` 保留了 Swift/SwiftUI 版本草稿，后续安装完整 Xcode 后可以继续迁移成更现代的 SwiftUI 工程。

## 产品路线图

### 已完成

- 眼睛休息、站立提醒、番茄钟和自定义分钟/秒。
- 眼睛提醒和站立提醒独立计时，并避免多个休息窗口互相抢占。
- 全屏休息页，支持完成、稍后、跳过、延长。
- 休息窗口默认不强制置顶，可在设置页或菜单栏开启置顶强提醒。
- 默认休息页可让开：非置顶模式下点击卡片外背景或切到其他 App，会隐藏休息页并继续本轮休息计时。
- 多种整体风格：极简呼吸、松弛森林、像素窗边、软糖玩具、夜间护眼。
- 风格同时影响设置窗口和休息窗口。
- 熄屏/唤醒、屏幕睡眠、锁屏/解锁后自动结算过期休息，避免休息窗口卡住主屏幕。
- 休息窗口状态看门狗：自动修复窗口残留、窗口类型不一致、唤醒后窗口丢失或层级不对等状态。
- 恢复后连续复查：唤醒、解锁或屏幕变化后会在 1 秒、3 秒、8 秒自动复查休息页状态，减少系统刚恢复时漏校准。
- 恢复诊断：菜单栏记录最近一次睡眠、锁屏、解锁或屏幕变化后的处理结果，保留最近 20 条事件链并跨应用重启保留，可一键复制详细诊断文本，也可手动运行恢复自检。
- 一键冒烟测试：脚本可覆盖构建安装、单实例、设置窗口、外部自动化链接和恢复压测，方便每次迭代后快速确认关键路径。
- 暂停态清屏：暂停提醒时会主动关闭休息页和残留全屏窗口，避免暂停后仍挡住主屏幕。
- 今日轻量统计：眼睛完成、站立完成、稍后、跳过。
- 本机统计面板：展示最近一周休息完成趋势、近 30 天摘要、30 天热力图、跳过率、站立总时长、连续天数、自动化策略占比、CSV 导出和 JSON 备份。
- 多策略统计：统计手动完成、只发通知、自动暂停会话和自动暂停时长，并写入 CSV/JSON 导出。
- 统计面板趋势解释：根据本周/上周变化、跳过率、站立时长和连续天数给出轻量反馈。
- 月度视图摘要：近 30 天热力图旁展示活跃率、日均完成次数和最佳日。
- 数据备份与恢复：可导出/恢复本地 JSON，包含当前设置、自动化关键词和近 30 天休息统计。
- 菜单栏显示模式：完整、只看眼睛、只看站立、极简图标、智能轮换。
- 菜单栏信息密度优化：智能轮换模式会平时短显示，休息中或临近提醒时展开倒计时。
- 菜单栏快速节奏：可直接切换 20-20-20、番茄 25/5 和调试 10 秒，并显示当前命中的节奏。
- 工作模式/轻打扰模式：会议、演示或专注时可从菜单栏开启，只保留系统通知和计时，不弹全屏休息页。
- 自动轻打扰模式：前台应用命中会议、演示、视频或游戏白名单时，自动抑制全屏休息页。
- 日历会议联动：授权读取日历后，当前会议进行中会自动进入只发通知模式。
- 日程关键词联动：可分别配置日程通知关键词和日程暂停关键词，录制、直播、演讲、面试等场景会自动暂停计时。
- 全屏/演示联动：检测到系统全屏、隐藏 Dock 或隐藏菜单栏状态时，自动进入只发通知模式。
- 安静时段：自动化设置页可设置固定时间段，在该时间内自动只发通知、不弹全屏休息页。
- 外部自动化链接：注册 `songyixia://`，可由快捷指令或其他工具触发轻打扰、快速节奏、暂停、立即休息、打开设置和恢复压测。
- 专注联动脚本模板：菜单栏可复制快捷指令、Hammerspoon 和 Raycast 示例，快速把系统专注或自定义热键接到轻打扰。
- 自动化诊断：可复制自动化状态、常用联动链接、前台应用和策略数量，方便排查外部工具联动。
- 自动化设置页：可开关自动策略、查看当前前台应用/日程命中状态，并编辑/恢复应用和日程关键词。
- 自动化设置页精简：主页面保留状态、场景开关、安静时段和高级入口，应用/日程关键词移入独立编辑弹窗。
- 更细的自动化策略：按应用选择只发通知、自动暂停或不处理，按日程关键词选择只发通知或自动暂停；不处理优先级最高，适合修正误命中。
- 登录时自动启动。
- 休息页风格增强：每个风格有专属提示和更明显的背景细节。
- 休息页增加动作建议轮换：眼睛休息、站立提醒、番茄休息会给出不同建议。
- 休息页应急出口：全屏休息页可按 Esc 跳过当前提醒，菜单栏也提供“应急关闭休息页”来清理异常残留窗口并重启计时。
- 非置顶休息页让开：点击背景或卡片空白区域的第一下就会隐藏休息页，本轮休息计时继续；只有开启“置顶强提醒”时才会固定在最前面。
- 休息页按钮自检：休息页展示或唤醒校准时会重新绑定完成、稍后、跳过和延长按钮，发现按钮链路异常会自动重建休息页。
- 恢复压测：菜单栏可运行多轮恢复压测，连续校准休息页、按钮链路、残留窗口和屏幕挂载状态，并写入可复制的恢复诊断。
- 午休恢复压测：可模拟长时间离开后站立休息已经过期，验证自动结算、休息页关闭和下一轮站立计时重置。
- 睡眠隐藏恢复压测：可模拟睡眠/锁屏时休息页被隐藏但本轮休息未过期，验证该恢复时会恢复、已让开时不会异常弹回。
- 长离开恢复压测：可模拟睡眠/锁屏很久后眼睛和站立休息都已过期，验证两个提醒自动结算、休息页清理和下一轮计时重置。
- 显示恢复压测：可模拟休息页位于屏幕外，验证显示器切换或外接屏断开后会回到当前屏幕内。
- 显示边界压测：可模拟休息页仍在屏幕内但尺寸继承旧显示器，验证会重新贴合当前屏幕并重排内容。
- 窗口让开压测：可模拟非置顶休息页、设置页和用户切走场景，验证休息页让开后计时继续、设置页不被关闭。
- 窗口层级压测：可验证设置页保持普通窗口层级、普通休息页让开后不弹回，且只有“置顶强提醒”才使用高层级。
- 自动化策略压测：可模拟安静时段和自动暂停，验证只发通知、关闭休息页和顺延提醒时间。
- 演示策略压测：可模拟全屏/演示轻打扰，验证到点只发通知、已有休息页会关闭、恢复自检不会重新弹窗。
- 日历策略压测：可模拟日历会议只通知和日程暂停，验证不弹休息页、暂停会关闭休息页并顺延提醒时间。
