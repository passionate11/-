# 松一下

松一下是一个轻量 macOS 菜单栏休息提醒工具，覆盖眼睛远眺、站立活动和番茄休息。

## 功能

- 眼睛休息提醒：默认 20-20-20，使用电脑 20 分钟后，提醒眺望 6 米外至少 20 秒。
- 站立提醒：默认每 2 小时提醒一次，站立 20 分钟。
- 眼睛提醒和站立提醒分别独立计时。
- 支持番茄钟预设：25 分钟专注，5 分钟休息。
- 设置窗口支持直接输入分钟和秒。
- 菜单栏可实时显示眼睛提醒和站立提醒，也支持智能轮换的短文案模式。
- 支持暂停、立即眼睛休息、立即站立、重新开始全部计时。
- 支持独立设置窗口，可详细配置两个提醒、通知和提醒窗口。
- 设置窗口采用更接近 macOS System Settings 的左侧导航布局。
- 到点可弹出全屏休息页，并发送系统通知。
- 支持多种整体风格：极简呼吸、松弛森林、像素窗边、软糖玩具、夜间护眼。
- 风格会同时影响设置窗口和全屏休息页。
- 显示设置页支持风格预览，切换风格时可以先看到休息页的大致气质。
- 支持菜单栏显示模式：眼睛 + 站立、只显示眼睛、只显示站立、极简图标、智能轮换。
- 支持登录时自动启动。
- 设置窗口包含本机 7 天休息趋势、近 30 天摘要、30 天热力图、跳过率、站立总时长、连续天数、CSV 导出和 JSON 备份。
- 设置会自动保存到 `UserDefaults`。

## 构建

```bash
scripts/build_app.sh
```

构建产物会生成到：

```text
outputs/EyeRest.app
```

## 运行

```bash
open outputs/EyeRest.app
```

打开后点击菜单栏里的眼睛图标，选择 `打开设置...` 可以进入详细设置窗口。

如果 macOS 提示来自未验证开发者，可以在 Finder 中右键打开，或到系统设置里允许打开。

## 安装到应用程序

```bash
scripts/install_app.sh
```

脚本会先构建，再覆盖安装到：

```text
/Applications/松一下.app
```

## 说明

当前可运行版本使用 Objective-C + AppKit 实现，以便在只有 Command Line Tools、没有完整 Xcode 的环境中构建。`Sources/EyeRest/main.swift` 保留了 Swift/SwiftUI 版本草稿，后续安装完整 Xcode 后可以继续迁移成更现代的 SwiftUI 工程。

## 产品路线图

### 已完成

- 眼睛休息、站立提醒、番茄钟和自定义分钟/秒。
- 眼睛提醒和站立提醒独立计时，并避免多个休息窗口互相抢占。
- 全屏休息页，支持完成、稍后、跳过、延长。
- 多种整体风格：极简呼吸、松弛森林、像素窗边、软糖玩具、夜间护眼。
- 风格同时影响设置窗口和休息窗口。
- 熄屏/唤醒、屏幕睡眠、锁屏/解锁后自动结算过期休息，避免休息窗口卡住主屏幕。
- 休息窗口状态看门狗：自动修复窗口残留、窗口类型不一致、唤醒后窗口丢失或层级不对等状态。
- 今日轻量统计：眼睛完成、站立完成、稍后、跳过。
- 本机统计面板：展示最近一周休息完成趋势、近 30 天摘要、30 天热力图、跳过率、站立总时长、连续天数、CSV 导出和 JSON 备份。
- 统计面板趋势解释：根据本周/上周变化、跳过率、站立时长和连续天数给出轻量反馈。
- 数据备份：可导出本地 JSON，包含当前设置和近 30 天休息统计。
- 菜单栏显示模式：完整、只看眼睛、只看站立、极简图标、智能轮换。
- 菜单栏信息密度优化：智能轮换模式会平时短显示，休息中或临近提醒时展开倒计时。
- 登录时自动启动。
- 休息页风格增强：每个风格有专属提示和更明显的背景细节。
- 休息页增加动作建议轮换：眼睛休息、站立提醒、番茄休息会给出不同建议。
- 设置页视觉重构：从顶部分段切换改为左侧导航 + 右侧内容区。
- 设置页增加原生列表行层级：眼睛、站立和显示方式页面使用行背景与分隔线组织控件。
- 显示方式页增加实时风格预览。

### 下一批优先级
